# ReSeeder

**Моментальная раздача торрентов без перехеширования**

[**Сайт**](https://reseeder.app) · [**Скачать**](https://reseeder.app) · [**Скриншоты**](https://reseeder.app/screenshots.html) · [English](README.md)

</div>

---

**ReSeeder** — бесплатная Windows-утилита для мгновенной раздачи уже скачанных фильмов и сериалов на **Kinozal**, **RuTracker** и других трекерах, без перехеширования. Находит уже скачанные файлы на диске и ставит их на раздачу за **секунды**. Без перехеширования, без повторного скачивания.

> ℹ️ Этот репозиторий — **витрина проекта**: исходный код приложения здесь не публикуется. Загрузка и подробности — на **[reseeder.app](https://reseeder.app)**.

## Возможности

- ⚡ **Моментальный выход на раздачу (cross-seeding)** — возвращает уже скачанные файлы в раздачу за секунды, без повторной проверки хешей.
- 🔎 **Находит переименованные файлы** — сопоставляет файлы с торрентами даже после переименования, хешируя только первый и последний кусок.
- 📦 **Раздаёт то, что есть** — раздаёт реально лежащие на диске файлы, без перекачивания.
- 🔗 **Один файл — несколько трекеров** — один локальный файл можно раздавать сразу на несколько трекеров.
- 🗂️ **Автосортировка по трекерам** — сам раскладывает раздачи по трекерам.
- 🗺️ **Карта раздачи** — наглядная карта пиров, которым вы раздаёте.
- 🛠️ **Для трекеров** — поддержка пасскея и API для интеграции с трекерами.
